mySome function checks if an array contains certain values that pass the conditional set by the callback function. In my example, imagine you are applying for jobs, yet you want a position that offers double what you are currently making. In the first box enter several salaries from different jobs. The second box will be the amount that you are making. mySome takes in any callback function that returns a boolean. In this example, the callback function I use doubles the salary, inputted by the user. If any of the values are equal or more than the amount means there is at least one job that meets the criteria
